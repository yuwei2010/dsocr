# Installation

## Stable release

To install dsocr, run this command in your terminal:

```sh
uv add dsocr
```

Or if you prefer to use `pip`:

```sh
pip install dsocr
```

## From source

The source files for dsocr can be downloaded from the [Github repo](https://github.com/yuwei2010/dsocr).

You can either clone the public repository:

```sh
git clone https://github.com/yuwei2010/dsocr
```

Or download the [tarball](https://github.com/yuwei2010/dsocr/tarball/main):

```sh
curl -OJL https://github.com/yuwei2010/dsocr/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd dsocr
uv sync
```
